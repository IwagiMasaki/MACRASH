## ストーリー
修学旅行、それは誰かに想いを告げる絶好の機会。

枕投げ、それは己の強さを証明するうってつけの機会。

主人公である真倉桜は、他の女に負けない強さの証明のため、枕と布団の山へ飛び込んでいく…！

## ゲームルール
相手が投げてくる枕を躱し、時には受け止めながら相手に枕を当てるゲーム。
先に３回枕を当てられてしまった方が寝てしまい、負けとなってしまう。
消灯まで６０秒…その間に決着をつけろ！
